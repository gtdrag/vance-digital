**src/db/queries/stats.ts** — Network statistics queries:
- `getNetworkStats(db: Kysely<DatabaseSchema>)`: Returns object with:
  - `totalSites`: COUNT from sites WHERE status = 'live'
  - `totalMemes`: COUNT from memes WHERE verified = 1
  - `totalDomains`: COUNT from domains WHERE status != 'failed'
  - `totalSpentUsd`: SUM(amount_usd) from transactions (all time)
  - `monthlySpentUsd`: SUM(amount_usd) from transactions WHERE created_at >= first day of current calendar month (Claude's discretion: use calendar month for simplicity)
  - `pendingDeployments`: COUNT from deployments WHERE status = 'pending'
- Return type: `NetworkStats` interface (export it)
- Handle empty database gracefully (return 0 for all counts, 0.0 for amounts)
- Use Kysely's `fn.countAll()` and `fn.sum()` for type-safe aggregation

**src/cli/commands/status.ts** — Status command:
- `statusCommand(options: { json?: boolean; quiet?: boolean })`:
  1. Create logger from options
  2. Load config with `loadConfig()` to get budget cap
  3. Get database with `getDb()`, call `getNetworkStats(db)`
  4. If `--json` flag: output `JSON.stringify(stats, null, 2)` to stdout and return
